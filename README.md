# DevToys.ImageHasher

Image Hasher (PHash/DHash/AHash/WHash) for [DevToys](https://devtoys.app/) 2.0.

## Installation

1. Download the `DevToys.ImageHasher` NuGet package from [NuGet.org](https://www.nuget.org/packages/DevToys.ImageHasher).
1. In DevToys, open `Manager Extensions`, click on `Install` and select the downloaded NuGet package.
